## About

This component has wave in universe Canvas. This components build with ThreeJS
and reac-fiber-three.

The animation responds to the equation, where:

- `a` is amplitude
- `x`is time
- `d1`is the duration of the animation
- `P0`is the position of a point

<img
  src="https://res.cloudinary.com/dzh1db41l/image/upload/v1613002462/arctan_fshyzx.png"
  alt="equation"
  style={{ width: "100%", maxWidth: 450 }}
/>

For demo, I simulate the position variation of a only point.

If the above function will not use `arctan` function this on values close to 0, the animation will fire like below:

<img
  src="https://media.giphy.com/media/bnHUTUYeKkwrXEyClH/giphy.gif"
  alt="sin"
/>

For this reason use the `arctan` function to smooth the animation.
